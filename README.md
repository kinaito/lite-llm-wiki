## 🚀 功能介绍
### 核心能力
- 📝 **知识卡片生成**：自动提取笔记核心内容，生成标准化知识卡片，包含概念、定义、关键信息
- 🏷️ **实体识别关联**：自动识别笔记中的实体概念，生成统一格式的实体页面，自动建立关联关系
- 📦 **批量处理支持**：支持单文件、多选文件、整个文件夹批量生成，高效处理存量笔记
- 🔄 **远程配置同步**：LLM配置、免费次数、系统公告实时同步，无需升级插件即可享受最新配置
- 🔒 **安全激活机制**：专业版采用RSA加密激活，支持设备绑定，安全可靠

### 支持的大模型
- 美团 LongCat 系列
- DeepSeek 系列
- OpenAI GPT 系列
- 所有兼容 OpenAI 接口格式的大模型

## 📥 安装方法
### 手动安装（当前推荐）
1. 下载本页最新 Release 中的 `lite-llm-wiki-vX.X.X.zip` 压缩包
2. 解压到 Obsidian 库的插件目录：`你的库路径/.obsidian/plugins/lite-llm-wiki/`
3. 重启 Obsidian，在「设置 → 第三方插件」中启用「Lite LLM Wiki」
4. 打开插件设置页面，配置知识卡片输出目录即可使用

### 官方市场安装（待审核通过后支持）
1. 打开 Obsidian「设置 → 第三方插件 → 社区插件市场」
2. 搜索「Lite LLM Wiki」
3. 点击「安装」，安装完成后启用即可

## 💡 使用方法
1. **基础配置**：打开插件设置，配置「知识卡片输出目录」和「实体页面输出目录」
2. **配置LLM服务**：填写你的大模型API信息（Base URL、API Key、模型ID），点击验证确认配置正确
3. **生成卡片**：
   - 单文件：右键笔记文件 → 「生成知识卡片」
   - 多文件：多选多个笔记文件 → 右键 → 「批量生成知识卡片」
   - 文件夹：右键文件夹 → 「批量生成文件夹内所有笔记的知识卡片」
4. **查看结果**：生成的卡片会保存到你配置的输出目录中，自动关联原笔记链接
