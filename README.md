# Lite LLM Wiki - Obsidian 知识卡片自动生成插件
> 当前最新版本：v1.0.9 | 最近更新：2026-04-30

## 插件简介
基于大语言模型的Obsidian知识管理插件，自动将你的Markdown笔记转换为标准化的知识卡片和实体页面，一键构建个人知识图谱，让你的知识沉淀更高效。

### 核心特性
- 📇 智能生成结构化知识卡片，自动梳理笔记核心内容
- 🤖 自动提取笔记中的核心概念和实体，生成标准化实体页面，自动构建双向关联
- ⚡ 支持单文件/多文件/文件夹批量生成，增量更新不覆盖原有内容
- 🎯 开箱即用，无需配置任何LLM服务，直接就能用
- 🔒 完全本地运行，绝不修改原始笔记文件，不写入任何FrontMatter，所有生成内容独立存放

## 快速上手
### 安装方式
#### 方式1：Obsidian 插件商店安装（推荐）
1. 打开Obsidian → 设置 → 第三方插件 → 关闭安全模式
2. 搜索「Lite LLM Wiki」，点击安装即可
3. 安装完成后启用插件

#### 方式2：手动安装
1. 在 [GitHub Releases](仓库地址) 下载最新版本的 `lite-llm-wiki-v1.0.9.zip` 压缩包
2. 打开Obsidian插件目录：
   - 打开Obsidian → 设置 → 第三方插件 → 在「已安装插件」小标题的右侧，找到文件夹📂样式的图标，点击即可直接打开插件目录
   - 手动路径通常为：`你的库目录/.obsidian/plugins/`（Mac）或 `你的库目录\.obsidian\plugins\`（Windows）
3. 将下载的压缩包直接放到插件目录下，右键解压到当前文件夹，会自动生成 `lite-llm-wiki` 文件夹
4. 删除原压缩包，重启Obsidian，在第三方插件中启用「Lite LLM Wiki」即可

### 使用方法
1. **首次配置**：打开插件设置，仅需配置两个目录路径：
   - 知识卡片输出目录：生成的知识卡片存放目录
   - 实体库目录：生成的实体页面存放目录
2. **生成卡片**：在文件浏览器中选择要处理的内容，右键选择「生成知识卡片」，支持三种选择方式：
   - 选中单个Markdown文件
   - 选中多个Markdown文件
   - 选中整个文件夹
3. **查看进度**：右上角会显示实时生成进度，完成后会通知结果
4. **浏览知识**：
   - 生成的卡片和实体会自动建立双向链接，点击实体名称即可查看关联的所有卡片
   - 打开Obsidian官方预装的「知识图谱」插件，选择你的知识卡片输出目录，即可看到完整的可视化知识图谱结构

## 常见问题 FAQ
> 遇到问题先看这里，90%的问题都能找到答案

### Q1：生成卡片需要网络吗？
A：默认使用内置公共API时需要网络，你也可以在设置中配置自己的LLM API地址（支持OpenAI/Anthropic/本地部署模型等），就可以完全私有化使用。

### Q2：会修改我的原始笔记文件吗？
A：绝对不会！插件只会读取你的原始笔记，所有生成的内容都存放在你配置的输出目录中，不会修改任何原始文件，请放心使用。

### Q3：免费版和专业版有什么区别？
A：免费版每天可以生成10次，专业版激活后可以享受更高的生成限额和更多高级功能。

### Q4：为什么生成的卡片内容不符合预期？
A：可以在设置中自定义提示词模板，根据你的笔记风格调整提示词，获得更符合你需求的生成结果。
